如何編譯：
於此目錄下使用 make 指令即可。

如何輸出 LLVM IR：
java -cp ./antlr-3.5.3-complete.jar:. myCompiler_test <測試用檔名.c> > <測試用檔名.ll>

如何執行：
lli < <測試用檔名.ll>

額外的功能：
printf 支援1-無限多參數。
if-else、for、while 支援三者混用和多層巢狀，同時亦可任意使用單行或區塊{}。

測試用檔案：
test1.c - 測試基本 assignment 及 for + while 迴圈。
test2.c - 測試忽略註解及多層巢狀 if-else + while + for 單行及區塊混用。
test3.c - 測試四則運算規則和括號優先計算，以及 printf 字串及任意參數數量。

支援的語法：
1.型態：int
2.數值：整數
2.函式：printf(支援1-無限多參數，第一個須為字串，其餘可為變數或運算式)
3.if, if else (支援混用單行及區塊{}，且支援巢狀。)
4.while (支援單行及區塊{}，且支援巢狀)
5.for (支援單行及區塊{}，且支援巢狀)
6.break, continue
7.註解：單行註解、區塊註解
8.字串
9.計算與邏輯(支援負號，可用括號()表示優先計算，運算子之運算順序符合 C 定義)：*、/、%、+、-、<<、>>、<=、>=、<、>、==、!=、&、|、&&、||
